# IKI

IKI is many things.
Among others a Clojure CLI.

## Installing

To install IKI,

1. Clone the repo

        git clone https://github.com/teodorlu/iki-clj.git
        
2. then install the script with [babashka/bbin][babashka-bbin]

        bbin install ./iki-clj/

[babashka-bbin]: https://github.com/babashka/bbin
